# Getting Started

Welcome to IRPF. This guide explains how to set up the framework for development and experimentation.

## Requirements
Install:

- .NET 8 SDK or newer  
- Visual Studio 2022 or VS Code  
- Windows 10/11  
- CodeWalker.API running locally  

Optional:

- Git  
- PowerShell
